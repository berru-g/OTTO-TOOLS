Ouvrez un terminal (ou invite de commandes) dans le dossier contenant le fichier .sh.

Donnez les droits d'exécution au script en utilisant la commande suivante :

Sur Linux/Mac :

bash

chmod +x create_project.sh

Sur Windows (avec Git Bash ou Cygwin) :

bash

chmod +x create_project.sh

Exécutez le script en tapant la commande suivante :

bash

./create_project.sh
